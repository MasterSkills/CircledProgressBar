# CircledProgressBar
一个简单的圆形进度条，通过简单的圆弧绘制来实现，实现了基本的大小适配、文字大小修改、颜色调整等功能。

# 属性说明

1、**FinsihedColor**
设置进度达到最大值后对勾的颜色

2、**TopColor**
设置已达成的进度条的颜色

3、**BottomColor**
设置底部整个进度条的颜色。

4、**Font**
设置进度值的显示字体

5、**ForeColor**
设置进度值的显示颜色
